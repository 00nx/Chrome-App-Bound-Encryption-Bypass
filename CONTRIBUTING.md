# Contributing Guidelines

Thank you for your interest in contributing to this project!  
This tool is intended **only** for security research, education, and authorized testing.  
By contributing, you agree to follow these guidelines and the [Code of Conduct](CODE_OF_CONDUCT.md).

---

## 1. Legal & Ethical Requirements
- Only contribute features or code that can be **useful, readable, and affects the program**
- Do **not** submit functionality intended for malicious deployment or unauthorized access.
- All code should be tested in a **controlled, authorized environment**.
- Ensure your contributions comply with local laws and regulations.
